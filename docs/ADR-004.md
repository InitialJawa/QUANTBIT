# ADR-004: Data Status Transparency System
**Status:** Accepted
**Date:** 2026-06-21

## Context
Data gratis (Yahoo Finance) memiliki keterbatasan akurasi dan coverage. User perlu tahu apakah data yang ditampilkan real-time, cached, atau estimated.

## Decision
Implementasi enum `DataStatus` (LIVE/CACHED/STALE/ESTIMATED) yang melekat pada setiap data point, ditampilkan sebagai badge visual di UI.

## Consequences
- **Positif:** Transparansi penuh, user bisa menilai trustworthiness data
- **Negatif:** Overhead komputasi dan UI tambahan
