# ADR-001: AI Provider — Google Gemini 2.5 Flash
**Status:** Accepted
**Date:** 2026-06-21

## Context
Aplikasi membutuhkan AI untuk menghasilkan executive summary naratif dari output kalkulasi engine kuantitatif. Diperlukan provider yang hemat biaya, cepat, dan memiliki konteks cukup besar.

## Decision
Menggunakan Google Gemini 2.5 Flash sebagai primary AI provider, dengan Groq dan OpenRouter sebagai fallback.

## Consequences
- **Positif:** Biaya rendah, latency cepat, context window besar
- **Negatif:** Perlu GEMINI_API_KEY, fallback diperlukan untuk redundancy
