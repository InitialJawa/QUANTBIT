# ADR-005: AI Context Persistence + DOX Framework
**Status:** Accepted
**Date:** 2026-06-22

## Context
Project dengan 100+ file, banyak keputusan arsitektur, dan pergantian AI agent membutuhkan sistem persistence agar konteks tidak hilang antar sesi.

## Decision
Mengadopsi AI Context Persistence System + DOX Framework:
- `docs/` — menyimpan state, decisions, onboarding
- `handover/` — snapshot tiap sesi
- Child AGENTS.md — DOX framework per folder durable

## Consequences
- **Positif:** Zero context loss antar sesi, AI baru bisa langsung kerja
- **Negatif:** Maintenance docs jadi tanggung jawab rutin
