# ADR-003: Database — Cloudflare D1 + SQLite
**Status:** Accepted
**Date:** 2026-06-21

## Context
Dibutuhkan database yang bisa jalan di edge (production) dan lokal (development) dengan biaya minimal.

## Decision
Menggunakan Cloudflare D1 (SQLite-compatible) untuk production dan better-sqlite3 untuk development lokal. Schema tunggal di `db/schema.sql`.

## Consequences
- **Positif:** Zero-cost untuk skala kecil, SQLite familiar, edge-native
- **Negatif:** Tidak ada replication bawaan, limitasi concurrent writes di D1
